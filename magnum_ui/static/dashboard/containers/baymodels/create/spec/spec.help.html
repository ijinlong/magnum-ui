Description:

Specify the specs for the nodes in the Bay.

Image: The name or UUID of the base image to customize for the bay.

Flavor: The nova flavor id to use when launching the bay. Default: m1.small

Master Flavor: The nova flavor id to use when launching the master node of the bay. Default: m1.small

Docker Volume Size: The size in GB for Docker Volume. Default: 25

Keypair: The name or UUID of the SSH keypair to load into the Bay nodes.
